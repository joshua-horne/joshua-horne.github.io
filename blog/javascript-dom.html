The Document Object Model

This week we looked at JavaScript and the DOM (Document Object Model)

The DOM is what JavaScript uses to interact with your website. In this model the structure of your web page should be viewed like a tree, with it beginning with a base that is then branched out into a number of different elements

In this example the base of your tree is the "Document", which refers to the entire page and all its content. It will then branch into sections like the header and body, and those sections will further branch out into parts like titles, headings etc.

With this strcucture in mind JavaScript is able to call on and interact with specific parts of your page.

So how can wee use JavaScript and the DOM? Let's work with something simple like a button. And we want this button to display a message when clicked. The button itself can be made through HTML and CSS, it's just an element of the page like any other. The interactivity is where JavaScript steps in.

First we need to call on the object for JavaScript to interact with, normally this is done by assigning the object in question a unique ID in the HTML and then calling that specific ID with JavaScript.

Once JavaScript knows what it's interacting with we can create a function to run when a specific event happens. In this example the function will be to display a message, and the event will be when the button is clicked.

To recap on this example; JavaScript calls the object (the button), creates and assigns a function (display message) and waits for an event (when clicked). When the event occurs, the function is run.

Now let's look at something a little more complex. Continuing with our button let's have it change the background colour of the page with each click, with a set order of colours being cycled through.

The object and event remain the same but we want to change the function. Rather than display a message we want to change background color. And because we want more than just one colour we will be using an array. An array is like a list or set.

Let's have the button cycle through the 7 colours of the rainbow, then our array will consist of 7 items (Red, Orange, Yellow etc.) The act of 'cycling' through these items can be achieved with a loop.

On each cycle or loop the of function (change background color) we will use a different item from the array e.g on 1st click we will change background to the 1st item of the array (red), 2nd click will change to 2nd item (orange)

See how our loop keeps track of how many times the function has been run and adjust the function accordingly.

Using these examples as a very simple base you can see how JavaScript and it's DOM bring interactivity to your website.
