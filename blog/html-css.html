Margin Border and Padding

Margin, Border and Padding are what makes up the CSS Box Model; a concept that allows us to add borders to our elements as well as define how we utilize the space between and within them.

Let’s take a closer look at each of these parts.

Margin: This is the space around an element. Think of it as the elements own little force field, creating a barrier between itself and other elements on the page. By increasing the size of the margin you increase the gap between other elements.

You can specify which sides of the element you want changed with margin-top, margin-right, margin-bottom and margin-left.

Border: This outlines the element. Like the frame around a picture, it is often used to clearly display elements to the user and can be styled in a number of ways.

You can set things like width, style and color. And as above you can also set which side of the element will be altered.

Padding: This is the space between the border and the content of the element. Using this we can position our content within the element where we want it.

Once again you can specify whether this affects all sides or only top, right etc.
